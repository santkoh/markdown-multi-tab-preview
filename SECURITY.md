# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.4.x   | Yes       |
| < 0.4   | No        |

## Reporting a Vulnerability

Please report security vulnerabilities via GitHub's
[private vulnerability reporting](https://github.com/santkoh/markdown-multi-tab-preview/security/advisories/new).

We will acknowledge receipt within 48 hours and aim to provide a fix within 7 days for critical issues.

**Please do not open public issues for security vulnerabilities.**

## AI-Generated Code Disclosure

This extension is developed using AI-assisted coding (Claude Code + Codex).
All generated code undergoes automated security scanning (CodeQL)
and cross-model review before merging.
